# GitHub Pages kurulum notu

Bu klasör doğrudan yayımlanabilir. Ek bir tema, Jekyll kurulumu veya paket yöneticisi gerektirmez.

## Kurulum

1. GitHub'da `KULLANICI-ADIN.github.io` adlı herkese açık bir repository oluştur.
2. Bu klasörün içindeki bütün dosyaları repository'nin ana dizinine yükle.
3. Dosyaları `main` branch'ine commit et.
4. **Settings → Pages** bölümüne gir.
5. **Deploy from a branch** seçeneğini seç.
6. Branch olarak `main`, klasör olarak `/ (root)` seç ve kaydet.
7. Site birkaç dakika içinde `https://KULLANICI-ADIN.github.io/` adresinde açılır.

## İçerik güncelleme

- Ana sayfa: `index.html`
- Hakkında: `about.html`
- Araştırmalar: `research.html`
- Yayınlar: `publications.html`
- Dersler: `teaching.html`
- CV: `cv.html`
- İletişim: `contact.html`
- Tasarım: `assets/css/style.css`

CV sayfasındaki **Print / save as PDF** bağlantısı tarayıcının yazdırma menüsünü açar ve sayfayı PDF olarak kaydetmeye izin verir.

## Uygulanan yayın ilkesi

R&R aşamasındaki çalışmaların dergi adları yazılmıştır. Yalnızca under review olan çalışmaların dergi adları sitede ve web CV'sinde gösterilmemiştir.

## Gizlilik

Telefon numarası kamuya açık siteye eklenmemiştir. E-posta adresi görünür durumdadır.
